# LoRA from Scratch

Low-Rank Adaptation (LoRA) and QLoRA for parameter-efficient fine-tuning, from scratch in PyTorch.

## Features

Low-rank decomposition
- 4-bit NF4 quantization
- adapter composition
- merge/unmerge
- memory analysis

## Tech Stack

PyTorch, NumPy, matplotlib
